How to use this tracker

Click the State, Region, or Status column header to sort. “Active” means a live application or package-level RFA/RFP is open now. “Recently closed” means a recent cycle just ended but is still useful for partner mapping. “Forecasted” means an official state source has announced an upcoming window or release period. “Monitor” means the state is strategically relevant, but no public competitive dates were found on official RHTP pages as of the verification date.

★ indicates the state row was updated from the prior weekly version.

Tip: click State, Region, or Status once for ascending order and again for descending order.
