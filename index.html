How to use this tracker

Use the search and dropdown filters to isolate states fast. “Active” means a live application, solicitation, or clearly open package. “Recently closed” means a recent cycle just ended but is still useful for partner mapping. “Forecasted” means an official source points to a coming window, webinar, or rollout stage. “Monitor” means the state is strategically relevant, but no public competitive dates were posted on official sources reviewed as of the verification date.

To show what changed without using stars, rows added or materially updated in this refresh carry the blue Updated this refresh tag and can be isolated with the Updated only filter.

Tip: click State or Region once for ascending order and again for descending order.
